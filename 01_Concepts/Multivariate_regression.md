\# Multivariate Regression



* In univariate regression we work on scenarios where in we only have one dependent and one independent variable.
* For example: If we're trying to predict \*\*Study Hours\*\* using \*\* Exam score\*\* of students, \*\*house size\*\* using \*\*house price\*\*, we're essentially implementing univariate regression.
* So, here we have one cause->one effect.



\## In \*\*Multivariate Regression\*\* we can predict outcome using many predictors.

* for instance, if we want to \*\* Predict the price of a house from: 1) Size(x1) and  2)No. of bedrooms(x2)\*\*
* The equation for above case would look like this: \*\* y= w0 + w1x1 + w2x2 \*\*, where w0 is y intercept, w1 and w2 are weights of each predictor.
* So, essentially in multivariate regression we find the best fitting flat surface in higher dimensions.
* W1, w2 are also known as coefficients which just reflects how important is each predictor(variable).
