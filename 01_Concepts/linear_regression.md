# Linear Regression

Linear Regression is a type of machine learning used to **predict numbers**.

It finds a **line** that best fits the data points.

**Formula:**  
y = w * x + b  
- y = predicted value  
- x = input value  
- w = weight (slope)  
- b = bias (intercept)  

**Example:**  
Predicting house price from size. The model finds a line that best predicts price from size.

**Applications:**  
- House price prediction  
- Sales forecasting  
- Stock price prediction  
- Any situation where you want to predict a number from other numbers
